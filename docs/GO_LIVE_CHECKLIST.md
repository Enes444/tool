# Go-Live Checklist

## A) Produkt/Scope
- [ ] Finale IA: Workspace (intern), Portal (extern)
- [ ] Feature Freeze v1.0 dokumentiert

## B) Technik
- [ ] `.env` vorhanden und `SPONSOR_OPS_JWT_SECRET` gesetzt
- [ ] CORS Origins korrekt
- [ ] Rate Limit aktiv
- [ ] API + Web Container starten sauber

## C) Sicherheit/Betrieb
- [ ] Backup erfolgreich
- [ ] Restore erfolgreich
- [ ] Runbooks aktuell

## D) Onboarding Assets
- [ ] Kunden-Admin SOP
- [ ] Kickoff Agenda
- [ ] FAQ / Troubleshooting

## E) Tenant Setup (pro Kunde)
- [ ] Kunden-Org angelegt
- [ ] Kunden-Admin angelegt
- [ ] 1 Test-Deal + Deliverables erstellt
- [ ] Portal-Link getestet

## F) Abnahme
- [ ] Login + Session stabil
- [ ] Tickets/Claims/Deliverables Flows grün
- [ ] Archive/Restore + Include Archived grün

## G) Live Tag
- [ ] Deploy + Smoke Test auf Live-URL
- [ ] Kunde eingeladen und erfolgreich eingeloggt
- [ ] Supportkanal aktiv

## H) Hypercare
- [ ] tägliche Triage (Woche 1–2)
- [ ] P0/P1 Bugs nach SLA
