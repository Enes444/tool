# Kunden-Admin SOP

## Täglich (15–30 Min)
1. Inbox öffnen
2. Tickets beantworten
3. Claims prüfen (approve/deny/paid)
4. Deliverables reviewen (approve/request changes)
5. Überfällige Fälle priorisieren

## Wöchentlich (60 Min)
- Offene Tickets > 7 Tage prüfen
- Claims-Aging prüfen
- Abgeschlossene Fälle archivieren
- Kurzreport: offen/abgeschlossen/blockiert

## Monatlich (90 Min)
- Rollen/Rechte überprüfen
- Inaktive Nutzer entfernen
- Backup/Restore Status gegenchecken
- Prozessverbesserungen aus wiederkehrenden Tickets ableiten
